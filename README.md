# Introduction to Robotics Final Project


### Project Proposal Ideas:

* Multi-target prioritization system
    * If multiple objects appear, robot chooses one based on rules(Could be voice):
        * closest to center
        * biggest
        * highest confidence class priority
    * shows algorithm design

* Autonomous cone / obstacle avoidance
    * Place obstacles in a small course and make the car navigate around them using the camera.
        * introduces planning or reactive navigation
        * looks like autonomy, not remote control
    * Keep it simple:
        * detect obstacle left/right
        * steer to open side
        * re-center afterward

* Visual docking / parking system

* Object-following car ( Voice guided)
